# exe Halftone Tool

Browser-based halftone image converter optimized for DTF (Direct to Film) printing.

**Features:**
- Multiple grid types: Square, Hex, Radial, Stochastic (FM)
- 12 dot shapes: Circle, Square, Diamond, Triangle, Hex, Cross, Star, Ring, Line, Flower, Heart, Slash
- DTF-optimized presets (Vintage, Standard, Photo, Stochastic)
- Colored halftone dots (original image colors)
- Original-Blend mode (smooth transition between halftone and full image)
- Texture effects: Grain, Dirt, Scratches, Dust, Splatter, Worn, Vignette
- Background removal with color picker
- PNG + SVG export with custom DPI/size
- Light/Dark mode
- 100% client-side — no data leaves your browser
